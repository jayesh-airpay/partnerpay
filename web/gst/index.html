Start Filing GSTR-1

To know more about how to file GSTR-1, keep reading further

1. Import / Create your data

Get your data into ClearTax GST software by following any of the four ways:

  • Create invoices on ClearTax GST software
  • If you are using Tally, export GSTR-1 under statutory reports & import the file into Cleartax GST
  • If you are using government GSTR-1 offline excel utility, fill the data & import the file into ClearTax GST
  • In other cases, you can use ClearTax excel template, fill the data & import it.

2. Prepare Invoice Summary

Government requires taxpayers to provide details of all types of transactions.

  1. Invoice level details are required for B2B, B2C Large, Exports, Credit Debit note (registered as well as unregistered) transaction. It will be auto populated based on the data prepared in previous step.
  2. For Advance received, Advance adjusted, Nil rated and B2C Others transactions, only consolidated values are required:
    1. If you are using Tally Export or Government GSTR-1 utility, these values will be auto-filled.
    2. If you are using ClearTax excel template or using ClearTax billing solution, these values need to be entered manually.

Note:from next filing, we will enable ability to auto-fill consolidated values of all transactions where invoices are created on ClearTax billing or have been imported using ClearTax excel template.

3. Prepare Invoice Series Data

Government also requires summary of documents issued during the tax period.It is a mandatory step. Here you need to give details of nature of document issued eg Credit note, invoice for outward supplies etc, serial number for eg INV 2005 to INV 2899, total number of documents issued, cancelled etc.

4. Prepare HSN Data

Government requires summary of all products sold /services rendered identified by their unique HSN/SAC codes. Only consolidated data is needed and not transaction level data. You can enter this directly on the corresponding section.

5. Enter turnover details

Enter amount of turnover of previous year & turnover for the period April-June 2017.(This is an one time activity).

6. Upload data to GSTN

Upload to GSTN is not same as filing return. As per government rule every supplier needs to upload the details of his supplies to GSTN. The same data will be made available by GSTN to your customer who will download it at time of filing their GSTR 2 & it will become part of his GSTR2A.

Note:

  • Uploading the data that you prepared in previous steps is a mandatory step.
  • It is advisable to upload transaction level data multiple times throughout the month. This will help to avoid last minute reconciliation troubles.

7. Download GSTR-1 Report

  • Once data is successfully uploaded, you need to download the GSTR-1 report generated by GSTIN. Review this to ensure that all your data is accurately uploaded and processed by GSTN/Government.

8. Freeze your GSTR-1 Data

  • After reviewing, if everything seems fine, you can freeze & submit the return by clicking on “Freeze & Submit” button.

Please note that once return is submitted, no further modifications will be allowed.

9. E-Sign & File GSTR-1

  • When you file your return, you need to e-verify your return by affixing a digital signature (DSC) or you can e-sign using your Aadhaar based mobile OTP.

As of now only DSC based signing is allowed through APIs. for Aadhaar based e-sign, visit GSTN portal at www.gst.gov.in

  • After signing, click on “File Return” button. Once return is successfully filed, you will get success message along with ARN number.
